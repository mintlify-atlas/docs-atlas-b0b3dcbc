## Get Started with SentimentAPI

This guide will walk you through analyzing your first customer comment using SentimentAPI. You'll learn how to authenticate, navigate to the sentiment analysis panel, and interpret AI-generated results.

<Steps>
  <Step title="Access the Application">
    Navigate to the SentimentAPI login page. If you're viewing the homepage, click the **"Acceder al Panel"** button in the top navigation or **"Comenzar a Analizar Gratis"** in the hero section.
    
    <Note>
      First-time users should click "IR AL DASHBOARD" to access the login page.
    </Note>
  </Step>

  <Step title="Authenticate with Your Credentials">
    Enter your email and password on the login form. SentimentAPI uses secure authentication with the following flow:

    **Request Format:**
    ```json
    {
      "email": "your-email@example.com",
      "contrasenia": "your-password"
    }
    ```

    **API Endpoint:**
    ```bash
    POST {NEXT_PUBLIC_API_URL}/login
    Content-Type: application/json
    ```

    **Response:**
    ```json
    {
      "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
    }
    ```

    Upon successful authentication, you'll receive a session token that's automatically stored in an HTTP-only cookie with:
    - **Duration**: 2 hours
    - **Security**: Secure flag enabled in production
    - **SameSite**: Lax

    <Warning>
      If you see "No autenticado" or "Tu sesión ha expirado", your token has expired. Return to the login page to re-authenticate.
    </Warning>
  </Step>

  <Step title="Navigate to Sentiment Analysis Panel">
    After successful login, you'll be redirected to `/sentiment` - the main analysis dashboard. You'll see:
    
    - **Panel de Análisis** header
    - Input field for entering comments
    - "Analizar Sentimientos" button
  </Step>

  <Step title="Analyze Your First Comment">
    Enter a customer comment in the text input field. For example:
    
    ```text
    This product exceeded all my expectations!
    ```
    
    Click **"Analizar Sentimientos"** to submit for analysis.

    **Behind the scenes**, your comment is formatted and sent to the API:

    ```typescript
    // Request payload format
    const dataToAnalyze = [
      { texto: "This product exceeded all my expectations!" }
    ];

    // API call
    fetch(`${process.env.NEXT_PUBLIC_API_URL}/sentiment`, {
      method: "POST",
      headers: { 
        "Content-Type": "application/json",
        "Authorization": `Bearer ${token}`
      },
      body: JSON.stringify(dataToAnalyze)
    });
    ```

    <Note>
      The button text changes to "Procesando..." while the AI analyzes your comment.
    </Note>
  </Step>

  <Step title="View and Interpret Results">
    Once processing completes, the **"Resultados del Modelo"** section appears with a table showing:

    **Response Format:**
    ```typescript
    interface SentimentAnalysisResult {
      texto: string;              // Your original comment
      prevision: 'POSITIVO' | 'NEGATIVO';  // AI classification
      probabilidad: number;       // Confidence score (0.0 - 1.0)
    }
    ```

    **Example Response:**
    ```json
    [
      {
        "texto": "This product exceeded all my expectations!",
        "prevision": "POSITIVO",
        "probabilidad": 0.976
      }
    ]
    ```

    **Understanding the Results:**

    | Column | Description | Example |
    |--------|-------------|----------|
    | **Comentario** | The original text you submitted | "This product exceeded..." |
    | **Prevision** | Sentiment classification with badge | 😊 POSITIVO (green) or ☹️ NEGATIVO (red) |
    | **Probabilidad** | AI confidence score as percentage with visual bar | 97.6% |

    <CardGroup cols={2}>
      <Card title="Positive Sentiment" icon="face-smile">
        Green badge with smile icon indicates positive customer feedback. High probability (>90%) means high confidence.
      </Card>
      <Card title="Negative Sentiment" icon="face-frown">
        Red badge with frown icon signals negative feedback requiring attention. Prioritize high-probability negatives.
      </Card>
    </CardGroup>
  </Step>
</Steps>

## Example: Analyzing Negative Feedback

Let's analyze a negative comment to see how the system identifies dissatisfaction:

<CodeGroup>
```text Input
No superó mis expectativas, muy decepcionante.
```

```json Response
[
  {
    "texto": "No superó mis expectativas, muy decepcionante.",
    "prevision": "NEGATIVO",
    "probabilidad": 0.924
  }
]
```
</CodeGroup>

The AI correctly identifies this as **NEGATIVO** with **92.4% confidence** - flagging this comment for immediate follow-up by your support team.
