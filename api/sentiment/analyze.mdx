## Endpoint

```
POST {NEXT_PUBLIC_API_URL}/sentiment
```

Analyzes the sentiment of text comments and returns predictions with confidence scores.

## Authentication

This endpoint requires a valid JWT token obtained from the `/login` endpoint.

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
  
  Format: `Bearer {token}`
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`
</ParamField>

## Request Body

The request body must be an array of objects, each containing a text field:

<ParamField body="texto" type="string" required>
  The text comment to analyze for sentiment
</ParamField>

### Request Structure

```typescript
[
  { texto: string }
]
```

## Response

Returns an array of `SentimentResult` objects:

<ResponseField name="texto" type="string">
  The original text that was analyzed
</ResponseField>

<ResponseField name="prevision" type="string">
  The predicted sentiment classification
  
  Possible values: `"POSITIVO"` | `"NEGATIVO"`
</ResponseField>

<ResponseField name="probabilidad" type="number">
  Confidence score of the prediction (0.0 to 1.0)
  
  Example: `0.95` represents 95% confidence
</ResponseField>

### Response Interface

```typescript
interface SentimentResult {
  texto: string;
  prevision: 'POSITIVO' | 'NEGATIVO';
  probabilidad: number;
}
```

## Status Codes

- **200 OK**: Sentiment analysis completed successfully
- **401 Unauthorized**: Missing or invalid authentication token
- **500 Internal Server Error**: Server error during processing

## Examples

<CodeGroup>

```bash cURL
curl -X POST "https://api.example.com/sentiment" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." \
  -d '[
    { "texto": "This product is amazing!" }
  ]'
```

```typescript TypeScript
const dataToAnalyze = [
  { texto: "This product is amazing!" }
];

const response = await fetch(`${process.env.NEXT_PUBLIC_API_URL}/sentiment`, {
  method: "POST",
  headers: { 
    "Content-Type": "application/json",
    "Authorization": `Bearer ${token}`
  },
  body: JSON.stringify(dataToAnalyze),
});

const results: SentimentResult[] = await response.json();
console.log(results);
```


```typescript

  { texto: "Great service and fast delivery!" },
```

## Usage Notes

- The `probabilidad` field is a decimal between 0 and 1 (multiply by 100 for percentage)
- Each request can analyze one or multiple comments
- Text should be properly encoded in UTF-8
- Empty or whitespace-only text may produce unexpected results
