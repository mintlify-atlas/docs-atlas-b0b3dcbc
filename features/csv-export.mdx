## Export from Statistics Dashboard

The statistics dashboard export includes both summary statistics and detailed comment data:

### Export Implementation

```typescript statistics/page.tsx
const handleExportCSV = (e: React.MouseEvent<HTMLButtonElement>) => {
  e.preventDefault();

  if (!dataResponse || !dataResponse.content) {
    setMessage("No hay datos para exportar");
    return;
  }

  try {
    setDowloading(true);

    // Define headers
    const headers = ["ID", "   Comentario", "     Prevision", "    Probabilidad", "    Fecha Registro "];
    
    // Create summary section
    let csvContent = `\n RESUMEN DE ANALISIS DE LOS ULTIMOS ${dataResponse.total_en_pagina} COMENTARIOS \n\n `;
    csvContent += ` Total en pagina, ${dataResponse.total_en_pagina}\n`;
    csvContent += ` Positivos, ${dataResponse.positivos}\n`;
    csvContent += ` Negativos, ${dataResponse.negativos}\n\n\n`;
    
    // Add column headers
    csvContent += headers.join(",") + "\n";

    // Map comment data
    const rows = dataResponse.content.map((item: any) => {
      const comentarioLimpio = item.comentario.replace(/\n/g, " ").replace(/"/g, '""');
      
      return [
        item.id,
        `"${comentarioLimpio}"`,
        item.prevision,
        item.probabilidad,
        `"${item.fecharegistro}"`
      ].join(",");
    });

    csvContent += rows.join("\n");

    // Create blob and download
    const blob = new Blob(["\ufeff" + csvContent], { type: "text/csv;charset=utf-8;" });
    const url = URL.createObjectURL(blob);
    const link = document.createElement("a");
    
    link.setAttribute("href", url);
    link.setAttribute("download", `reporte_sentimientos_${new Date().getTime()}.csv`);
    link.style.visibility = 'hidden';
    
    document.body.appendChild(link);
    link.click();
    document.body.removeChild(link);
    
    setMessage("CSV generado localmente con éxito");
  } catch (err) {
    console.error("Error al generar CSV:", err);
    setMessage("Error al generar el archivo");
  } finally {
    setDowloading(false);
  }
};
```

## Export from Bulk Processing

After uploading and processing a CSV file, you can export the results:

### API-Based Export

```typescript upload-csv/page.tsx
const handleExportCSV = async (e: React.MouseEvent<HTMLButtonElement>) => {
  e.preventDefault();
  setDowloading(true);
  
  if (!token) {
    setMessage("Sesión expirada. Por favor, inicia sesión nuevamente.");
    setUploadStatus('error');
    return;
  }

  try {
    const response = await fetch(
      `${process.env.NEXT_PUBLIC_API_URL}/sentiment/export-csv`, 
      {
        method: "POST",
        headers: { 
          "Authorization": `Bearer ${token}`,
          "Content-Type": "application/json"
        },
        body: JSON.stringify(resultados),
      }
    );

    if (!response.ok) {
      throw new Error(`Error ${response.status}: No se pudo generar el CSV`);
    }

    // Convert response to blob
    const blob = await response.blob();
    
    // Create temporary URL and download
    const url = window.URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = "resultados_sentimiento.csv";
    document.body.appendChild(a);
    a.click();
    
    // Cleanup
    a.remove();
    window.URL.revokeObjectURL(url);
    setDowloading(false);
    setMessage('¡Excelente! El archivo se exportó correctamente.');
    setUploadStatus('success');
  } catch (err: any) {
    setUploadStatus('error');
    if (err.message === "Failed to fetch") {
      setMessage("No se pudo conectar con el servidor.");
    } else {
      setMessage(err.message);
    }
  }
};
```

## CSV Features

### UTF-8 BOM Encoding

Exported files include UTF-8 BOM (`\ufeff`) for proper character encoding:

```typescript
const blob = new Blob(["\ufeff" + csvContent], { type: "text/csv;charset=utf-8;" });
```

<Tip>
This ensures special characters (accents, tildes, etc.) display correctly in Excel and other spreadsheet applications.
</Tip>

### Text Cleaning

Comment text is automatically cleaned before export:

```typescript
const comentarioLimpio = item.comentario
  .replace(/\n/g, " ")      // Remove line breaks
  .replace(/"/g, '""');     // Escape double quotes
```

### Timestamp-Based Filenames

Each export gets a unique filename with timestamp:

```typescript
link.setAttribute("download", `reporte_sentimientos_${new Date().getTime()}.csv`);
```

Example filename: `reporte_sentimientos_1710410400000.csv`

<Info>
Timestamped filenames prevent accidental overwrites when exporting multiple datasets.
</Info>

## Download Process

The export process is fully client-side for instant results:

1. **Generate CSV Content** - Format data as CSV string
2. **Create Blob** - Convert to binary large object with UTF-8 encoding
3. **Generate URL** - Create temporary object URL
4. **Trigger Download** - Programmatically click download link
5. **Cleanup** - Remove temporary DOM elements and revoke URL

<Warning>
Ensure your browser allows downloads from the SentimentAPI domain. Some browsers may block automatic downloads for security reasons.
</Warning>


## Troubleshooting

<AccordionGroup>
  <Accordion title="Export Button Disabled">
    **Cause**: No data available to export
    
    **Solution**: Run an analysis first (either bulk upload or statistics request) before attempting to export.
  </Accordion>
  
  <Accordion title="Special Characters Appear as Gibberish">
    **Cause**: Incorrect encoding in spreadsheet application
    
    **Solution**: 
    - In Excel: Use "Get Data" > "From File" > "From Text/CSV" and select UTF-8 encoding
    - The exported file includes UTF-8 BOM which should handle this automatically
  </Accordion>
  
  <Accordion title="Download Doesn't Start">
    **Cause**: Browser blocking automatic downloads
    
    **Solution**: 
    - Check browser's download settings
    - Allow pop-ups and downloads from the SentimentAPI domain
    - Try a different browser if the issue persists
  </Accordion>
  
  <Accordion title="Empty or Incomplete Export">
    **Cause**: Network error during export generation
    
    **Solution**: 
    - Check your internet connection
    - Retry the export
    - If using bulk export, ensure the API endpoint is accessible
  </Accordion>
</AccordionGroup>
