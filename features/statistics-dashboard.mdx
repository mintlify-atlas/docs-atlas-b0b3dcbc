## Overview

The Statistics Dashboard provides a comprehensive view of your sentiment analysis history. Access aggregate metrics, review past analyses, and export historical data for reporting and trend analysis.

<Info>
All sentiment analyses are automatically saved to the database, allowing you to review historical data and track sentiment trends over time.
</Info>

## Data Structure

The statistics API returns comprehensive data including both metrics and individual comment details:

```typescript statistics/page.tsx
// Individual comment result
interface SentimentResult {
  id: number;
  comentario: string;
  prevision: 'POSITIVO' | 'NEGATIVO';
  probabilidad: number;
  fecharegistro: string;
}

// Complete API response with statistics
interface SentimentApiResponse {
  total_en_pagina: number;
  positivos: string;        // Percentage as string
  negativos: string;        // Percentage as string
  content: SentimentResult[];  // Array of individual results
  totalElements: number;
  totalPages: number;
  number: number;           // Current page number
}
```

## Requesting Different Dataset Sizes

You can customize the number of recent comments to analyze using the size parameter:

```typescript statistics/page.tsx
const handleAnalyze = async (e: React.FormEvent) => {
  e.preventDefault();
  setLoading(true);

  const numComent = parseInt(comment);

  try {
    const response = await fetch(
      `${process.env.NEXT_PUBLIC_API_URL}/stats?size=${numComent}`, 
      {
        method: "GET",
        headers: {
          "Content-Type": "application/json",
          "Authorization": `Bearer ${token}`
        },
      }
    );

    const data = await response.json();
    setDataResponse(data);
    setResultado(data?.content);
  } catch (err) {
    // Error handling
  } finally {
    setLoading(false);
  }
};
```

## Results Table

The dashboard displays individual comments in an interactive table:

### Table Columns

1. **Comentario** - The original comment text with truncation for long content
2. **Prevision** - Color-coded sentiment badge (green for POSITIVO, red for NEGATIVO)
3. **Probabilidad** - Confidence score displayed as percentage with visual progress bar


## Use Cases

### Daily Monitoring

Review the last 100 comments each morning to catch urgent issues:

```bash
# Quick daily check
GET /stats?size=100

# Result: 82% positive, 18% negative
# Alert if negative % > 25%
```

### Weekly Reporting

Generate weekly reports for stakeholders:

```bash
# Weekly analysis (7 days × ~100 comments/day)
GET /stats?size=700

# Export to CSV for presentation
```

### Trend Analysis

Compare sentiment over different time periods:

```bash
# This week
GET /stats?size=700

# Last week (offset by current week's data)
GET /stats?size=1400  # Then compare first 700 vs last 700
```

### Campaign Impact Assessment

Measure sentiment before and after product launches or marketing campaigns:

```bash
# Post-campaign (last 1000 comments)
GET /stats?size=1000

# Compare to baseline historical data
```

## API Endpoint

Access statistics programmatically:

```bash
curl -X GET "https://api.sentimentapi.com/stats?size=500" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json"
```

<Info>
Authentication is required. Include your API token in the Authorization header.
</Info>
